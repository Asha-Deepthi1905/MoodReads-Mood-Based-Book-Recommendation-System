1. index.php (Homepage)
User enters mood like I feel nostalgic and hopeful in a textarea. On button click, JavaScript saves it in sessionStorage and redirects to results.php.
↓
2. results.php + script.js
JavaScript fetches the mood from sessionStorage, sends it via fetch() POST to recommend.php.
↓
3. recommend.php (PHP Backend)
PHP decodes mood input → uses strpos() to extract keywords → maps them to vibe_tags.
↓
4. SQL Query (MySQL)
Query joins books, book_vibe_tags, and vibe_tags to find matching books.
Books are returned randomly using ORDER BY RAND().
↓
5. JavaScript renders books
Loops over returned books[], builds HTML cards with book image, title, author, and heart icon ❤️.
↓
6. Favorite Button (Heart)
On heart click → fetch() POST to save_fav.php. PHP checks session, inserts into favorites.
↓
7. Book Detail Popup
On image click → calls book_detail.php?book_id=.... Displays full description and vibes in a modal.
↓
8. Mood Logging
Every mood is logged into mood_logs table with timestamp and user_id.
